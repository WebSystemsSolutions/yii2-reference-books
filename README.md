# yii2-reference-books
[RU]
```
Компонент справочников:
- позволяет динамически (через административный интерфейс) создавать произвольное количество справочников вида:
ключ ->  значение  +  доп. поля
- динамически прикреплять обьект значения справочника за полями  моделей  ActiveRecord
- значения кешируются
+  доп. плюшки
```
[ENG]
```
References component:
- allows you to dynamically (via the administrative interface) create an random number of directories of the form:
key -> value + additional fields
- dynamically attach the directory value object beyond the fields of ActiveRecord models
- values are cached
+ add. bonuses
```
